Image Steganography in C

Image Steganography in C is a simple project that hides secret text messages inside 24-bit BMP images using the Least Significant Bit (LSB) technique. The hidden message can later be extracted from the encoded image without visibly changing the image.
This project was developed to understand image processing, file handling, dynamic memory allocation, and bit manipulation in C.

Features
- Encode secret messages into BMP images
- Decode hidden messages from BMP images
- Uses LSB (Least Significant Bit) steganography
- Supports 24-bit BMP files
